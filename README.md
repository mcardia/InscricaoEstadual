# Validador de Inscrição Estadual

## Linguagem
Javascript

## Como usar
Inclua o arquivo ie.js em seu site e use da seguinte forma:

```javascript
if ( ! InscricaoEstadual.uf("SP").ie("123456789").valido() ) {
    alert("Inscrição Estadual Inválida");
}
```
